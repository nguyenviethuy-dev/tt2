## Usage

```tsx
import * as AvatarPrimitive from '~/components/primitives/avatar';

const GITHUB_AVATAR_URI = 'https://github.com/mrzachnugent.png';

function Example() {
  return (
      <Avatar.Root alt="Zach Nugent's Avatar">
        <AvatarImage source={{ uri: GITHUB_AVATAR_URI }} />
        <Avatar.Fallback>
          <Text>ZN</Text>
        </Avatar.Fallback>
      </Avatar.Root>
  );
}
```

## Props

### Root

Extends [`View`](https://reactnative.dev/docs/view) props

|  Prop   |  Type   |     Note     |
| :-----: | :-----: | :----------: |
| alt \*  | string  |              |
| asChild | boolean | _(optional)_ |

### Image

Extends [`Image`](https://reactnative.dev/docs/image) props except `alt`

|   Prop   |  Type   |     Note     |
| :------: | :-----: | :----------: |
| asChild  | boolean | _(optional)_ |

### Fallback

Extends [`View`](https://reactnative.dev/docs/view) props

|  Prop   |  Type   |     Note     |
| :-----: | :-----: | :----------: |
| asChild | boolean | _(optional)_ |
