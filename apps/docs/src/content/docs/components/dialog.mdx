A popup view that appears on top of the main view or another dialog, making the content underneath unclickable.

## How to add

Add the dialog primitives and the following code to your codebase.

### Add dialog primitive

Follow the steps to add the [dialog primitive](/primitives/dialog#how-to-add) to your codebase.

### Copy/Paste

Create a file `~/components/ui/dialog.tsx` and copy/paste the following code into that new file:

<Code
  lang='tsx'
  title = '~/components/ui/dialog.tsx'
  code = {importedCode} />

## Usage

```tsx
import {
  Dialog,
  DialogClose,
  DialogContent,
  DialogDescription,
  DialogFooter,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from '~/components/ui/dialog';

function DialogExample() {
  const [open, setOpen] = React.useState(false);
  return (
    <Dialog open={open} onOpenChange={setOpen}>
      <DialogTrigger>
        <Text>Edit Profile</Text>
      </DialogTrigger>
      <DialogContent className='sm:max-w-[425px]'>
        <DialogHeader>
          <DialogTitle>Edit profile</DialogTitle>
          <DialogDescription>
            Make changes to your profile here. Click save when you're done.
          </DialogDescription>
        </DialogHeader>

        <DialogFooter>
          <DialogClose>
            <Text>OK</Text>
          </DialogClose>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  );
}
```
